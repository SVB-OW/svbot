# SVBot

Python bot for setting up the ranked gauntlet

## Running

### Requirements

- Node.js: [https://nodejs.org/en/]

### Setup

- npm install

### Start

#### Normal run

```bash
npm start
```

#### Run with watch on save

```bash
npm run watch
```
